# Ebook Marketplace Logo Setup

Please copy your logo image to:
- `/home/thee1/ebookMP/ebook-marketplace/public/logo.png`

You should also create an Open Graph image (1200x630px) at:
- `/home/thee1/ebookMP/ebook-marketplace/public/og-image.png`

These images are used in:
1. Browser tab icon (favicon)
2. Header logo
3. Farcaster Frame preview
4. Social media sharing

After copying the logo, restart the development server with:
```bash
cd /home/thee1/ebookMP/ebook-marketplace
npm run dev
```
